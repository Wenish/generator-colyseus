# colyseus server (typescript)


## Dev Setup
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:2657
npm run dev
```

## Production Build
``` bash
# build for production with minification
npm run build

# build for production with minification and start your server
npm start

```
